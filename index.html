#1. Two Sum
Easy Arrays Hashing
Intuition

The brute force checks every pair in O(n²). But we can do better: for each number x, we need to know if target − x already exists. A hash map gives us O(1) lookup…

🔑 Key insight: store what you've seen, check what you need. One pass, no second loop.

Approach 2 — HashMap (optimal)
# O(n) time · O(n) space def twoSum(nums, target): seen = {} for i, n in enumerate(nums): diff = target - n if diff in seen: return [seen[diff], i] seen[n] = i
T: O(n) S: O(n)
